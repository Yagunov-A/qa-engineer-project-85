## Тестируемое приложение: 
**Codebattle** — игра с открытым исходным кодом, разрабатываемая сообществом Hexlet. Это платформа для дуэлей кодирования в реальном времени. Два игрока параллельно решают одну и ту же задачу, тот, кто решит ее первым, выигрывает матч.     
Расположение приложения https://codebattle.hexlet.io/   
Ссылка на Github https://github.com/hexlet-codebattle/codebattle.git
